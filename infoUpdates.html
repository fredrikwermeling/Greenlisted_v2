Updates

2026

Validate sgRNA feature / A new validation tool that lets you search any number of sgRNA sequences and find whether they are present in any of the embedded libraries. The validation index includes top-ranked sgRNAs from several widely used CRISPR knockout libraries sourced from Addgene and the Broad Institute GPP, including Brunello, Brie, GeCKO v2, Gattinara, Gouda, Jacquere, Julianna, VBC, Yusa Human v1, Yusa Mouse v2, TKOv3, and mTKO. The tool returns the matched gene symbol, library name, gene ID, and available on-target/off-target scores for each sequence. Separate indices are available for human and mouse libraries.

Jacquere (human) and Julianna (mouse) libraries / Added two new CRISPR knockout libraries from the Broad Institute/John Doench lab, designed with the latest on-target and off-target scoring (RS3 and CFD). These libraries replace Brunello and Brie as the recommended next-generation libraries for human and mouse screens, respectively. See Drepanos LM et al. Balancing off-target and on-target considerations for optimized Cas9 CRISPR knockout library design. bioRxiv 2025. doi: 10.1101/2025.08.26.672375 (link to preprint).

2025
